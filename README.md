# Basic Contact Form

A simple HTML form styled with CSS.

## HTML Tags Used
- `<!DOCTYPE html>`
- `<html>`
- `<head>`
- `<meta>`
- `<title>`
- `<style>`
- `<body>`
- `<div>`
- `<form>`
- `<fieldset>`
- `<legend>`
- `<label>`
- `<input>` (text, email, tel, number, date, radio)
- `<select>`
- `<option>`
- `<button>`
